# Athlete Assistant

Первый MVP помощника спортсмена без Firebase и без базы данных.

## Что уже есть

- расширенная анкета спортсмена: цель, вес, дата цели, питание, продукты, тренировки, сон, ограничения;
- главный экран с фокусом дня, готовностью, путем к цели и ближайшими действиями;
- блок питания: завтрак, обед, перекус, ужин;
- блок тренировок на 4 недели с учетом спорта, инвентаря, длительности и фидбека;
- фидбек после тренировки;
- рекомендации на основе анкеты;
- сохранение данных в браузере через `localStorage`;
- готовность к публикации через GitHub + Vercel.

## Важно

В этой версии нет аккаунтов и общей базы. Данные сохраняются только на устройстве пользователя. Если открыть приложение на другом телефоне или очистить браузер, профиль нужно будет заполнить заново.

## Как выложить бесплатно

1. Создай репозиторий на GitHub, например `athlete-assistant`.
2. Загрузи в него все файлы из этой папки.
3. Зайди на Vercel и выбери `Add New Project`.
4. Подключи GitHub-репозиторий.
5. Framework Preset оставь `Other`.
6. Build Command оставь пустым.
7. Output Directory оставь пустым.
8. Нажми `Deploy`.

После публикации Vercel даст ссылку вида:

```text
https://athlete-assistant.vercel.app
```

## Что добавлять дальше

1. Генерацию блюд из продуктов пользователя.
2. Закрытие упражнений по одному: вес, повторы, сложность.
3. Историю веса и тренировок.
4. API погоды.
5. API калорийности продуктов.
6. Позже, если понадобится, базу данных и вход в аккаунт.

## Как обновлять сайт

1. Замени файлы в GitHub на новую версию.
2. Сделай commit в ветку `main`.
3. Vercel сам опубликует обновление.
